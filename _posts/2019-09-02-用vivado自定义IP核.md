1. 在ip_repo下面自建一个目录，该目录为该ip核的工程目录。
2. 在该目录下设置一个src文件夹，编辑一个顶层文件。
3. create ip package
4. 编辑和生成ip核，综合看报告。注意在setting里选择不delele工程。
5. 将ip核的目录增加到原工程中。


PS：若是从ISE工程copy过来，可以用edif文件。生成办法，用ISE Design Suite xx Bit Command Prompt来生成。 ngc2edif xxx.ngc xxx.edf.
